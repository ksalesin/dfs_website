Readings

  • Jessica Riskin, "Machines in the Garden" — on the tension between technology and nature.
  • Donna Haraway, "A Cyborg Manifesto", blurring human, animal, and machine.
  • Karen Barad, "Meeting the Universe Halfway", how matter and meaning are entangled.
  • Jane Bennett, "Vibrant Matter", finding life in non-living things.
  • Tim Ingold, "The Textility of Making", craft as thinking with materials.

    • Adafruit: Adafruit manufactures a number of electronics products and produces a number of learning resources, including written tutorials, introductory videos for beginners.
    • Tiny Circuits: Tiny Circuits makes tiny open source electronics to be used to create miniature electronic projects.
    • Sparkfun: Sparkfun sells open-source hardware and has resources and tutorials on its website.
    • Shapeways: Shapeways is a 3D printing marketplace that has tutorials on creating your own file to be printed.
    • Thingiverse: Thingiverse is an open-source 3D printing library that includes tutorials, software, and printable files.
    • Tinkercad: Tinkercad lets you create 3D digital designs and has tutorials as well.
    • Sketchup: Sketchup is a 3D drawing tool that can help with creating 3D printing files.
    • Rhinoceros: Rhinoceros is a commercial 3D computer graphics and CAD software that also includes tutorials
